# Must-Port Build Configuration Patches

These 9 `.patch` files capture CryptPad's modifications to ONLYOFFICE core build
configuration files. They are required for the WASM/Emscripten compilation to succeed.

## How to Apply

```bash
# From tools/x2t-wasm/
./patches/apply-all.sh /path/to/vanilla-core
```

This applies stubs first, then all must-port patches, then harfbuzz.patch.
